| # | Пункт | Критерий | Наш сайт |

| 1 | Количество шрифтов | Не более 2-3 на сайт | 2 шрифта: EB Garamond (заголовки) + Lato (тело). В статьях — только Lato |

## Как пользоваться

Перед каждым релизом или крупным обновлением — пройтись по таблице. Особое внимание:
- Пункт 5 (сироты/вдовы) — проверять после любого изменения текстов
- Пункт 8 (контраст) — проверять при смене цветов, можно через webaim.org/resources/contrastchecker
- Пункт 11-12 (адаптивность) — тестировать на мобильном после правок в layout
- Пункт 21 (OG-теги) — после смены title/description проверить через developers.facebook.com/tools/debug

## Ключевые CSS-переменные

```css
--color-text: #0F172A          /* основной текст */
--color-text-muted: #475569    /* вторичный текст (WCAG AA: 5.1:1) */
--color-primary: #1E3A8A       /* синий бренда */
--color-accent: #D4AF37        /* золотой акцент */
--font-body-size: clamp(1rem, 0.43vw + 0.73rem, 1.125rem)  /* 16-18px */
--lh-body: 1.6                 /* межстрочное body */
--lh-heading: 1.3              /* межстрочное заголовков */
```

## Инструменты проверки

- Контраст: webaim.org/resources/contrastchecker
- OG-превью: developers.facebook.com/tools/debug
- Адаптивность: Chrome DevTools → Device Toolbar
- Типографика: браузер → Inspect → Computed → font-size, line-height
- Accessibility: Chrome DevTools → Lighthouse → Accessibility
